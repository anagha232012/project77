Console Project

Visual Studio code is a IDE where You can code websites. You can get Visual Studio Code by downloading it and opening a folder that you should credate before hand and selecting it.Once that is done, open a file by right clicking on the explorer and selecting file. Make sure the file names end in .html , .js and .css , unless you want to do something else like a svg. Then you can start coding.

Using the console is helpful because it tells you any mistakes in your code that you may have not caught. You can open console by doing inspect and click the small tab that is named console. The way you can show your code on the console is by writing console.log() in the HTML. In the brackets of the console.log() , you put a variable or whatever you want to print in the console.
